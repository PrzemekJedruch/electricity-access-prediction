# Predicting Global Electricity Access Using Machine Learning

In this project, I developed a machine learning pipeline to predict electricity access levels across countries using socioeconomic and technological indicators.

## Workflow

- Data cleaning
- Feature selection
- Correlation analysis
- Random Forest regression
- SHAP explainability
- Forecasting future electricity access

## Results

The Random Forest model achieved:
- R² Score: 0.86
- MAE: 5.80

The analysis showed that internet usage, education, and urbanization are strongly associated with electricity access.

## SHAP Analysis

SHAP values helped identify the most influential predictors in the model.

## Forecasting

The model was also used to forecast future electricity access levels for selected countries.
